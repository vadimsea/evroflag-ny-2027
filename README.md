# Еврофлаг — Новогодний лендинг 2027

Промо-лендинг корпоративных новогодних наборов компании [Еврофлаг](https://evroflag24.ru/).

## Запуск

```bash
npm install
npm run dev
```

Сборка:

```bash
npm run build
npm run preview
```

## Стек

- Vite + React + TypeScript
- [Framer Motion](https://github.com/motiondivision/motion) — анимации
- [Lenis](https://github.com/darkroomengineering/lenis) — плавный скролл
- [tsParticles](https://github.com/tsparticles/tsparticles) — снег
- [Lucide React](https://github.com/lucide-icons/lucide) — иконки
